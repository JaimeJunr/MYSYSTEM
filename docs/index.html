Homestead

Gerenciador de sistema em terminal — limpeza, instaladores e configuração Zsh

Go · Bubbletea · Charm

O que é

Homestead reúne scripts de limpeza, instaladores de ferramentas e o assistente de configuração Zsh numa única interface no terminal. Pensado para manter a máquina em ordem e configurar o shell em poucos passos.

Funcionalidades

  • Limpeza — Scripts para caches (Docker, npm, apt) e detecção de arquivos grandes
  • Monitoramento — Bateria e uso de memória
  • Instaladores — IDEs (Cursor, Claude Code) e ferramentas a partir de uma lista curada
  • Wizard Zsh — Passo a passo: Zsh, Oh My Zsh, Powerlevel10k, plugins e ferramentas (NVM, Bun, etc.)
  • TUI — Interface no terminal com Bubbletea e Lipgloss

Instalação

Opção 1 — Baixar binário (recomendado)

Linux (amd64 ou arm64). Os binários são gerados automaticamente em cada release. Baixe, dê permissão e coloque no PATH.

Baixar na página de Releases 
# Exemplo (arquivos: homestead-linux-amd64, homestead-linux-arm64)
chmod +x homestead-linux-amd64
sudo mv homestead-linux-amd64 /usr/local/bin/homestead
homestead

Opção 2 — Compilar do código

Requisitos: Go 1.21+ e Linux.

git clone https://github.com/JaimeJunr/Homestead
cd Homestead
make install

Ou execute direto: make run
