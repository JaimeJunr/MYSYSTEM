MySystem

Gerenciador de sistema em terminal — limpeza, instaladores e configuração Zsh

Go · Bubbletea · Charm

O que é

MySystem reúne scripts de limpeza, instaladores de ferramentas e o assistente de configuração Zsh numa única interface no terminal. Pensado para manter a máquina em ordem e configurar o shell em poucos passos.

Funcionalidades

  • Limpeza — Scripts para caches (Docker, npm, apt) e detecção de arquivos grandes
  • Monitoramento — Bateria e uso de memória
  • Instaladores — IDEs (Cursor, Claude Code) e ferramentas a partir de uma lista curada
  • Wizard Zsh — Passo a passo: Zsh, Oh My Zsh, Powerlevel10k, plugins e ferramentas (NVM, Bun, etc.)
  • TUI — Interface no terminal com Bubbletea e Lipgloss

Instalação

Requisitos: Go 1.21+ e Linux (Ubuntu/Debian recomendado).

git clone https://github.com/JaimeJunr/MYSYSTEM
cd MYSYSTEM
make install

Ou execute direto: make run

Uso rápido

↑/↓ ou j/k navegar · Espaço marcar · Enter confirmar · n ou → próximo · Esc voltar · q ou Ctrl+C sair
